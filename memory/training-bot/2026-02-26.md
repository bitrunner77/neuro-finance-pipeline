## 🧲 Skill: gifgrep

### What It Does
gifgrep is a CLI/TUI tool for searching GIF providers (Tenor and Giphy), downloading results, and extracting still frames or contact sheets from GIFs. It's designed for fast GIF discovery and processing workflows.

**Key Features:**
- Search GIFs from Tenor/Giphy APIs
- Browse results in an interactive TUI
- Download GIFs to local storage
- Extract still frames at specific timestamps
- Generate contact sheets (grid of sampled frames)
- JSON output for scripting/integration

---

## 📚 How to Use It

### Installation
```bash
# Via Homebrew (macOS)
brew install steipete/tap/gifgrep

# Via Go
go install github.com/steipete/gifgrep/cmd/gifgrep@latest
```

### Commands

**Search GIFs:**
```bash
gifgrep "cats" --max 5
gifgrep search --json "celebration" | jq '.[0].url'
```

**Interactive TUI:**
```bash
gifgrep tui "office handshake"
```

**Download:**
```bash
gifgrep "cats" --download --max 1 --format url
```

**Extract Still Frame:**
```bash
gifgrep still ./clip.gif --at 1.5s -o still.png
```

**Create Contact Sheet:**
```bash
gifgrep sheet ./clip.gif --frames 9 --cols 3 -o sheet.png
```

### Configuration
- `GIPHY_API_KEY` - Required for Giphy source
- `TENOR_API_KEY` - Optional (uses demo key if unset)
- `GIFGREP_SOFTWARE_ANIM=1` - Force software animation
- `GIFGREP_CELL_ASPECT=0.5` - Adjust preview geometry

## 🛠️ Project Built: gif-search.sh

Since gifgrep wasn't installed on this system and Go wasn't available, I built a bash script that replicates the core functionality using curl and ffmpeg.

### Features Implemented:
1. **Search** - Query Tenor API and display formatted results
2. **Download** - Save GIFs to ~/Downloads/gifs/
3. **Still Extraction** - Extract a single frame at specified time
4. **Sheet Creation** - Generate contact sheets with ffmpeg

### Files Created:
- `/root/.openclaw/workspace/projects/gif-search/gif-search.sh` - Main script
- `/root/Downloads/gifs/happy-jumping.gif` - Downloaded sample GIF (4.1MB)
- `/root/.openclaw/workspace/projects/gif-search/extracted-frame.png` - Single frame extraction (155KB)
- `/root/.openclaw/workspace/projects/gif-search/contact-sheet.png` - 3x2 grid contact sheet (364KB)

### Usage Examples:
```bash
# Search for GIFs
./gif-search.sh search "celebration" --max 3

# Download a GIF
./gif-search.sh download "https://media.tenor.com/xxx.gif"

# Extract a still frame at 1 second
./gif-search.sh still ./animation.gif --at 1.0 -o frame.png

# Create a contact sheet
./gif-search.sh sheet ./animation.gif --frames 6 --cols 3 -o overview.png
```
