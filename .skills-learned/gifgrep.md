# gifgrep

**Learned:** 2026-02-26

**Summary:** CLI/TUI tool for searching GIFs from Tenor/Giphy, downloading, and extracting stills/sheets.

**Key Commands:**
- gifgrep search "query" --max 5
- gifgrep tui "query"
- gifgrep still ./clip.gif --at 1.5s -o still.png
- gifgrep sheet ./clip.gif --frames 9 --cols 3 -o sheet.png

**Project:** Created gif-search.sh bash script as alternative implementation
